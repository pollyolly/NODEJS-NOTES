### Generate Key
Generate Key using Node Js
```
$node generate_key.js
```
or Generate Key with type PEM using CLI
```
$ssh-keygen -t rsa -m PEM -b 2048 -f license_key
$ssh-keygen -e -m PEM -b 2048 -f license_key > license_key_public.pem
```
